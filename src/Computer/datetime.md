# DateTime

ISO 8601 date and time format was first published in 1988. It has been superseded by a second edition ISO 8601:2000 in 2000 and by the current third edition ISO 8601:2004 published on 1 December 2004.

Date: 2018-01-22

Date and time in UTC:

- 2018-01-22T04:58:00+00:00
- 2018-01-22T04:58:00Z
- 20180122T045800Z

Week: 2018-W04
Date with week number: 2018-W04-1
Ordinal date: 2018-022

## Dates

```
YYYY-MM-DD  or  YYYYMMDD
YYYY-MM
--MM-DD  or --MMDD
```

## Times

```
hh:mm:ss.sss  or  hhmmss.sss
hh:mm:ss  or  hhmmss
hh:mm or  hhmm
hh
```

## Combined date and time representations

```
<date>T<time>
```

## Time intervals

```
<start>/<end>
<start>/<duration>
<duration>/<end>
<duration>
```
